API Documentation

Get started in minutes. Here are example requests for each endpoint.

Find Email

POST
curl -X POST https://api.example.com/api/find-email \ -H "X-API-Key: your-api-key" \ -H "Content-Type: application/json" \ -d '{ "domain": "google.com", "first_name": "John", "last_name": "Doe" }'

Verify Domain

POST
curl -X POST https://api.example.com/api/verify-domain \ -H "X-API-Key: your-api-key" \ -H "Content-Type: application/json" \ -d '{ "domain": "google.com" }'

Bulk Find

POST
curl -X POST https://api.example.com/api/bulk-find \ -H "X-API-Key: your-api-key" \ -H "Content-Type: application/json" \ -d '{ "domain": "google.com", "names": [ {"first_name": "John", "last_name": "Doe"}, {"first_name": "Jane", "last_name": "Smith"} ] }'

Get Patterns

GET
curl -X GET https://api.example.com/api/patterns/google.com \ -H "X-API-Key: your-api-key" # Response: { "success": true, "domain": "google.com", "patterns": [ {"pattern": "first.last", "commonality": "Very Common"}, {"pattern": "flast", "commonality": "Common"} ] }
